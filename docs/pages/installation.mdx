# Installation

Picasso can be installed via various methods. 

## Package Managers

### Homebrew

If you are using macOS or Linux and already have [Homebrew](https://brew.sh/) installed, getting Picasso is super easy.

```bash
brew install andersnormal/picasso/picasso
```
